# 🍔 McDonald's Sales Performance Dashboard (Excel Project)

## 📌 Project Overview

This project analyzes McDonald's sales data using Microsoft Excel to uncover key business insights.  
An interactive dashboard was built using Pivot Tables, Charts, Slicers, and Timelines to visualize sales performance.

The objective of this project is to demonstrate practical data analysis skills including:
- Data cleaning
- Data modeling
- Pivot table analysis
- KPI calculation
- Interactive dashboard design

## 📁 Dataset Description

The dataset consists of two main tables:

### 1️⃣ Menu Items Table
- Menu Item ID
- Item Name
- Category
- Price

### 2️⃣ Order Details Table
- Order ID
- Order Date
- Order Time
- Item ID
- Price
- Day Name
- Day Type (Weekend/Weekday)
- Time of Day
- Hour, Minute, Second
- Month & Month Index

Total Records: 12,000+ rows

---

## 📈 Key KPIs Included in Dashboard

- ✅ Total Revenue
- ✅ Total Orders
- ✅ Average Order Value (AOV)
- ✅ Most Ordered Item
- ✅ Least Ordered Item

---

## 📊 Analysis Performed

- Revenue by Category
- Revenue by Category across Months
- Top 5 Menu Items by Sales
- Order Volume by Hour (Peak Time Analysis)
- Sales Trends by Weekday vs Weekend
- Sales by Day of Week
- Average Items per Order

---

## 🎛 Interactive Features

- Category Slicer
- Date Timeline Filter
- Hour Slicer
- Dynamic Pivot Charts

All charts are interconnected using Report Connections.

---

## 💡 Key Insights

- Certain categories contribute significantly higher revenue despite lower order frequency.
- Sales peak during evening hours.
- Top 5 items contribute major share of total revenue.
- Weekend and weekday sales show stable trends.
- Average order value helps understand customer spending behavior.
